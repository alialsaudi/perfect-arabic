# Starter Prompts

Use any of the following as the first message in a chat inside the **Perfect Arabic — العربية السليمة** project. Replace the ellipsis (`…`) with your own text.

---

## Arabic — عربية

### فحص نصّ كامل
```
دقّق هذا النص نحوياً وفق النحو الوافي، واذكر القاعدة في كل تصويب:

…
```

### مراجعة مع الحفاظ على الأسلوب
```
راجع الفقرة التالية، ولا تغيّر الأسلوب إلا عند وجود خطأ نحوي أو صرفي. اذكر درجة التصحيح (واجب / أرجح / جائز):

…
```

### السؤال عن تركيب بعينه
```
هل يصح هذا التركيب في الفصحى؟ علّل من النحو الوافي:

«…»
```

### طلب إعراب
```
أعرب الجملة التالية مع بيان القاعدة من النحو الوافي:

…
```

### سؤال عن قاعدة
```
ما حكم المستثنى بعد «إلا» في الجملة المنفية؟ اذكر المسألة من النحو الوافي.
```

```
ما شروط إعراب الأسماء الستة بالحروف؟
```

```
متى يجب تأنيث الفعل مع الفاعل؟ ومتى يجوز؟
```

### التمييز بين نحو وصرف وإملاء
```
راجع النص التالي، وميّز بوضوح بين الأخطاء النحوية والصرفية والإملائية:

…
```

### عند التردد بين فصحى وعامية
```
هذا النص أقرب إلى العامية. أريد تحويله إلى فصحى سليمة وفق النحو الوافي:

…
```

## English

### Full check
```
Check the following Arabic text against Al-Nahw Al-Wafi. Preserve my
wording unless there is a grammar or morphology issue. For each correction,
give the rule, a short justification, and the severity
(واجب / أرجح / جائز):

…
```

### Single-rule question
```
Using Al-Nahw Al-Wafi, explain the rule for the accusative of specification
(التمييز) after numerals from 11 to 99. Cite the باب or مسألة.
```

### Parsing
```
Parse (إعراب) this sentence word by word, citing the relevant باب from
Al-Nahw Al-Wafi:

…
```

### Distinguishing categories
```
Review the following Arabic text. Separate your findings into:
- grammar (نحو)
- morphology (صرف)
- spelling (إملاء, flagged as notes, not errors)

…
```
